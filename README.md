# Full Stack JavaScript: Basics

Introductory JavaScript practice files covering core language primitives, operators, and control flow. Run examples with Node or open in a browser console.

## Table of Contents
- [01_datatypes_and_variables.js](01_datatypes_and_variables.js) — primitive types and variables
- [02_operators.js](02_operators.js) — arithmetic, comparison, logical, assignment operators
- [03_string.js](03_string.js) — string creation and common methods
- [04_numbers.js](04_numbers.js) — number handling and conversions
- [05.math_methods.js](05.math_methods.js) — `Math` utilities
- [06.challenge.js](06.challenge.js) — small practice challenge
- [07_if_else.js](07_if_else.js) — conditional statements
- [08_switch_statement.js](08_switch_statement.js) — switch-case control flow
- [09_user_inputs.js](09_user_inputs.js) — reading user input
- [10_rock_paper_scissors.html](10_rock_paper_scissors.html) — interactive game with HTML/CSS/JS
- [11_loops.js](11_loops.js) — for, while, and do-while loops
- [12_functions.js](12_functions.js) — function declarations and expressions
- [13_scope.js](13_scope.js) — variable scope and closures

## Usage
1. Install Node.js if you have not already.
2. From this folder, run a file with `node <filename>.js` (example: `node 02_operators.js`).
3. Edit and re-run to experiment with different inputs.
